#### chatmail-relay-themes

##### overview 
by default chatmail relays have a standard template which works fine, but it can be customized to your liking

##### usage  
- clone the repo: `git clone https://github.com/ccclxxiii/chatmail-relay-themes.git`
- backup your existing relay files in case you want to revert
- copy all the files from `themename/www/src` to your `relay/www/src` directory `cp 31337/www/src/* relay/www/src`
- re-run the deployment for your chatmail relay `cmdeploy run`
- refresh your chatmail relay website
